# SQL_retail_sales_project1
A quick analysis of a retail sales dataset with MySQL.

1. Database Setup - A database is created with a defined schema where the dataset will be stored. 
2. Table - A table 'retail_sales' is created where the data that has columns such as date, time, customer id, gender, age, etc.
3. Data Exploration - A quick glance is taken at the dataset to check the length of the dataset and study its columns.
4. Data Cleaning - The dataset is checked for null values and they are removed, if any.
5. Data Analysis - Data related to monthly and yearly sales, orders, shifts are analyzed.
